# Прогноз покупок в интернет-магазине

## Описание проекта
Этот проект предсказывает, совершит ли посетитель сайта покупку или нет. Мы анализируем поведение пользователя: сколько времени он провел на разных страницах, когда зашел на сайт (будни или выходные) и насколько близко сейчас праздники.

## Какие инструменты использовались
*   **Python** — основной язык.
*   **CatBoost** — библиотека для обучения модели.
*   **Pandas** — для работы с таблицами.
*   **Matplotlib / Seaborn** — для графиков.

## Что интересного в данных
*   **PageValues (Ценность страницы)** — самый важный фактор. Если пользователь заходит на страницы с высокой ценностью, он почти наверняка купит.
*   **Дисбаланс** — на сайте покупают редко (всего 15% заходов), поэтому мы настроили модель так, чтобы она не пропускала этих редких покупателей.

## Результаты
*   Качество модели (ROC-AUC) составило более **0.90**.
*   Модель хорошо видит разницу между "просто посмотреть" и "намерен купить".
*   В проекте есть файл `predict.py`, который позволяет быстро проверить любого нового пользователя.
